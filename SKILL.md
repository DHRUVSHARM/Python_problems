# Python Problems Project

This repository contains Python algorithm practice solutions. Prefer the dedicated project skills for repeat workflows:

- Use `$algorithm-summarizer` to explain the problem, approach, pattern, and complexity for a solution file.
- Use `$algorithm-dry-run-explainer` to create or add an example dry run for a solution file.

Both skills are project-local folders in this repository.
